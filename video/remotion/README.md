# GP-AI-Agent Pitch Video

This Remotion project renders the 1080p captioned hackathon-pitch video from
the six exported presentation slides and the matching speech transcript.

```bash
cd video/remotion
npm run studio
npm run render
```

The rendered MP4 is written to:

`output/video/GP-AI-Agent_Hackathon_Pitch_Captioned.mp4`

The video is intentionally silent and includes timed captions for the pitch
transcript. Record the provided narration yourself, or explicitly authorize a
third-party text-to-speech service before adding a synthesized voice track.
