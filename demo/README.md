# GP Agent demo

Vercel-ready Next.js showcase for the AMD Hackathon Track 1 agent.

```bash
npm install
npm run dev
```

The app works without credentials using representative demo responses. To enable live Fireworks answers, configure these Vercel environment variables:

- `FIREWORKS_API_KEY`
- `FIREWORKS_BASE_URL`
- `ALLOWED_MODELS`

Deploy with the Vercel project root set to `demo`, or run `vercel --cwd demo`.
